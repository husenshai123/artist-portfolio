# Fine Art Portfolio 🎨

A full-stack, responsive artist portfolio showcasing fine art with a dynamic backend for user interactions. 

🔗 **Live Demo:** [Click here to view the live website](https://sunilsurywanshi.vercel.app)

## 🛠️ Tech Stack
* **Frontend:** React.js (Vite)
* **Backend:** Node.js, Express.js
* **Deployment:** Hosted independently on Vercel

## ✨ Key Features
* **Interactive Gallery:** Clean, responsive grid layout to display high-quality artworks.
* **Working APIs:** Fully functional Contact Form and Guestbook integrated with a Node.js backend.
* **Mobile-First Design:** Optimized for all screen sizes.
